# Pacific Dance — Brand & Website Direction Deck

A client-facing brand-reveal deck for **Pacific Dance** (Irvine, CA), prepared for Tim, Lori, and Ian.
Built in the same format as the WildPath brand deck: a 16:9 HTML deck, keyboard-driven, print-to-PDF ready.

**▶ View the deck:** https://mghenry13.github.io/pacific-dance-brand-guide/

Use the on-screen arrows, or `←` / `→` / `space`, to move through the slides. Press `F` for fullscreen.

**37 slides in four parts** (navy separator slides between), built around **four client
questions** gut-checked mid-deck and locked on a final "Make the Calls" slide:
№ 1 the story (3 angles: Second Home / Pure Joy / Rec-to-Pro) · № 2 the style (4 palette
directions from the logo blues: Coastal Minimal / Royal Bold / Warm Spotlight / Gradient Stage) ·
№ 3 the typefaces (4 Squarespace-native pairings, Podkova · Inter recommended) ·
№ 4 the primary CTA (Book / Try / Start with / Get a Free Class — "Book" recommended).
Plus: audience, proof (incl. 2 review slides — 10 curated Google/Yelp quotes with a permission ask, from Notion "Website Reviews — Top Picks"), voice · the logo as fixed anchor · photography direction ·
**AI photo-editing before/afters** (building / studio / trophy wall, sources in Google Drive
"AI before and after") · site map (17 → 8) · conversion funnel · close.

Sources: the Notion working draft "Pacific Dance — Messaging, Brand, Site Map & Plan",
the owner interview, and the current-site archive one folder up (`../`).

## Notes

- The deck's own chrome dresses in **Direction 1 (Coastal Minimal)** — the direction closest
  to the logo — while presenting all four directions equally for the client to pick.
- Logo + core palette (black / royal blue / silver / white) are treated as fixed constraints;
  logo hexes `#00244C` · `#0072AA` · `#00A8E5`.
- All photos are real Pacific Dance images pulled from the site archive (`../assets/`).
  The AI-imagery slide shows *concepts to generate* using treated real photos as style
  references — no generated images are in the deck yet.
